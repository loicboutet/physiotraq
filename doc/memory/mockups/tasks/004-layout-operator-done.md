# Tâche: Layout Operator (Field User)

## Description
Créer le layout pour l'espace opérateur terrain avec navigation simplifiée.

## Éléments UI requis
- [ ] Header compact
  - Logo PhysioTraq (petit)
  - Navigation horizontale: Dashboard, Monitor, Treatments, Profile
  - Indicateur device actif
- [ ] Zone de contenu maximisée (monitoring = priorité)
- [ ] Footer minimal ou absent
- [ ] Mode "Monitor" plein écran possible

## Notes UX
- Interface épurée pour se concentrer sur le monitoring
- Gros boutons, texte lisible (usage terrain, potentiellement avec gants)
- Optimisé pour tablette/smartphone
- Couleurs d'alerte très visibles
