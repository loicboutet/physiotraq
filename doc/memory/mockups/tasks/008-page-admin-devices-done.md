# Tâche: Admin Devices List

## Description
Liste des devices enregistrés avec leur statut.

## Éléments UI requis
- [ ] Header avec titre + bouton "Add Device"
- [ ] Filtres: All / Online / Offline
- [ ] Table/Cards des devices
  - Nom du device
  - Serial number
  - Status (badge online/offline)
  - Dernière température
  - Assigné à (user)
  - Last seen
  - Actions (View, Edit, Delete)
- [ ] Empty state si pas de devices

## Notes UX
- Voir rapidement l'état de la flotte
- Tri et recherche si beaucoup de devices
- Code couleur pour statuts
