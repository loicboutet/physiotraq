# Tâche: Operator Dashboard

## Description
Dashboard simplifié pour l'opérateur terrain.

## Éléments UI requis
- [ ] Big CTA: "Start New Treatment" (très visible)
- [ ] Card: My Devices
  - Liste des devices assignés
  - Status de chaque device
  - Clic pour monitorer
- [ ] Card: Treatment in Progress (si existant)
  - Température actuelle
  - Durée
  - Patient info
  - Bouton "Resume Monitoring"
- [ ] Card: Recent Treatments
  - 5 derniers traitements
  - Quick stats

## Notes UX
- Action principale ultra visible
- Vue orientée "action" pas "admin"
- Accès rapide au monitoring
