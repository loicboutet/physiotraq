# Tâche: Admin Dashboard

## Description
Tableau de bord principal pour le propriétaire de licence.

## Éléments UI requis
- [ ] Card: License Status
  - Status badge (Active/Expired)
  - Date d'expiration
  - Bouton "Manage Billing"
- [ ] Card: Devices Overview
  - Nombre total de devices
  - Online vs Offline
  - Liste rapide des devices actifs
- [ ] Card: Treatments Today
  - Nombre de traitements du jour
  - En cours vs terminés
- [ ] Card: Team
  - Nombre de membres
  - Quick link vers gestion équipe
- [ ] Section: Recent Activity
  - Timeline des derniers événements

## Notes UX
- Vue d'ensemble en un coup d'œil
- Actions rapides accessibles
- Indicateurs visuels clairs (vert/rouge/orange)
