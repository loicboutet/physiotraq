# Tâche: Treatment Detail (Replay)

## Description
Vue détaillée d'un traitement passé avec replay de la courbe.

## Éléments UI requis
- [ ] Header: Session ID + Date
- [ ] Section: Patient Info
  - Nom ou "Anonymous Session"
  - Age, Sex, Weight
- [ ] Section: Summary
  - Device utilisé
  - Operator
  - Duration
  - Start/End time
  - Max temperature
  - Alerts triggered (39°C, 40°C)
- [ ] Section: Temperature Chart
  - Courbe complète du traitement
  - Zoom possible
  - Seuils marqués
- [ ] Section: Raw Data (collapsible)
  - Table des mesures
- [ ] Actions
  - Export CSV
  - Export XLS
  - Print

## Notes UX
- Revue post-traitement
- Données exportables pour analyse
