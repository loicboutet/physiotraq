# Tâche: Admin Device Detail

## Description
Page détail d'un device avec configuration et historique.

## Éléments UI requis
- [ ] Header: Nom device + status badge + actions (Edit, Delete)
- [ ] Section: Device Info
  - Serial number, MAC address
  - Firmware version (+ update available?)
  - Date d'enregistrement
- [ ] Section: Assignment
  - User assigné actuellement
  - Bouton pour changer l'assignation
- [ ] Section: WiFi Credentials
  - Liste des 4 slots WiFi configurés
  - Bouton pour modifier
- [ ] Section: Location (via IP)
  - Dernière localisation approximative
  - Mini carte si possible
- [ ] Section: Recent Treatments
  - Liste des derniers traitements sur ce device
- [ ] Section: Statistics
  - Nombre total de traitements
  - Temps d'utilisation

## Notes UX
- Vue complète du device
- Actions de configuration accessibles
