# Tâche: Admin Team Management

## Description
Gestion des membres de l'équipe (opérateurs).

## Éléments UI requis
- [ ] Header avec titre + bouton "Invite Member"
- [ ] Liste des membres
  - Avatar/initiales
  - Nom complet
  - Email
  - Role (Admin/Operator)
  - Devices assignés (count)
  - Status (active/pending invitation)
  - Actions (View, Edit, Remove)
- [ ] Modal/Form pour inviter un nouveau membre
  - Email
  - First name, Last name
  - Devices à assigner

## Notes UX
- Voir qui a accès
- Gérer facilement les assignations
- Invitation par email
