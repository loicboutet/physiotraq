# Tâche: Setup Mockups Namespace

## Description
Configurer le namespace `/mockups` avec les routes, contrôleurs de base et la page index qui liste toutes les pages mockées.

## Éléments à créer
- [ ] Route namespace `mockups` dans `routes.rb`
- [ ] `MockupsController` avec action `index`
- [ ] Vue `mockups/index.html.erb` - Page listant tous les mockups par profil
- [ ] Configuration Tailwind si nécessaire
- [ ] Import de la font DM Sans

## Notes techniques
- L'index doit lister les pages par catégorie (Auth, Admin, Operator)
- Liens cliquables vers chaque mockup
- Style cohérent avec le style_guide.html
