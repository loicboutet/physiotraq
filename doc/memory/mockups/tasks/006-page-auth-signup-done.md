# Tâche: Page Signup (Create Account + Buy License)

## Description
Page d'inscription avec achat de licence intégré.

## Éléments UI requis
- [ ] Section 1: Infos compte
  - First name, Last name
  - Email
  - Password, Password confirmation
- [ ] Section 2: Plan selection
  - Affichage du prix licence annuelle
  - Résumé de ce qui est inclus
- [ ] Section 3: Payment (Stripe Elements ou redirect)
  - Indication "Powered by Stripe"
- [ ] Bouton "Create Account & Subscribe"
- [ ] Lien "Already have an account? Sign in"

## Notes UX
- Processus en 1 page si possible (checkout simplifié)
- Ou wizard 2-3 étapes si trop long
- Sécurité visible (badges, https)
