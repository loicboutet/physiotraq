# Tâche: Layout Admin (License Owner)

## Description
Créer le layout pour l'espace admin (propriétaire de licence) avec sidebar et header.

## Éléments UI requis
- [ ] Sidebar fixe à gauche (280px)
  - Logo PhysioTraq
  - Navigation: Dashboard, Devices, Team, Billing, Treatments, Settings
  - Indicateur de licence (status, expiration)
  - Lien vers profil utilisateur
- [ ] Header sticky
  - Titre de la page courante
  - Breadcrumb si pertinent
  - Actions rapides (notifications?)
- [ ] Zone de contenu principale scrollable
- [ ] Responsive: sidebar collapse en mobile (hamburger menu)

## Notes UX
- Navigation claire et intuitive
- Couleurs du brand (orange pour les éléments actifs)
- Indicateurs visuels de l'état de la licence
